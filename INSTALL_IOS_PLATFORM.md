# Установка платформы iOS 26.1

Для сборки архива необходимо установить платформу iOS 26.1.

## Способ 1: Через Xcode GUI

1. Откройте **Xcode**
2. Перейдите в **Xcode → Settings** (или **Preferences**)
3. Откройте вкладку **Platforms** (или **Components**)
4. Найдите **iOS 26.1** и нажмите кнопку загрузки/установки
5. Дождитесь завершения установки

## Способ 2: Через командную строку

```bash
# Установка через xcodebuild (требуется sudo)
sudo xcodebuild -downloadPlatform iOS
```

## После установки

После установки платформы запустите сборку:

```bash
export TEAM_ID=972MD5K36E
./build_sidestore.sh
```

## Альтернатива: Сборка без архива

Если установка платформы невозможна, можно собрать проект через обычный build и создать IPA вручную, но это более сложный процесс.
